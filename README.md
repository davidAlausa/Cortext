# Cortext

**AI-powered multi-tenant chatbot platform**  
*Portfolio project demonstrating full-stack, AI, and cloud development skills.*

---

> **Note:** This project is built solely as a portfolio showcase. It is **not intended for commercial use**. Also, as this project is currently under development, this document may be subject to changes and for now does not reflect the fully functional state of the product.

## Project Overview

Cortext is a proof-of-concept platform that allows businesses (or demo users) to create AI chatbots based on their own documents. It features:

- Multi-tenant **employee/admin dashboard**
- Public **chatbot UI** embeddable on websites
- Document upload, storage, and vectorized search
- Retrieval-Augmented Generation (RAG) AI responses
- Support for multiple reasoning models 

---

## Architecture
- **Frontend:** Next.js (TypeScript, TailwindCSS)  
- **Backend:** FastAPI (Python) for AI, Next.js API for general business logic  
- **Databases:** Pinecone vector DB + AWS storage  
- **AI Models:** TBD
- **Hosting/Cloud:** AWS Amplify, S3, CloudFront, Cognito  

---

##  Features (MVP)

- Upload and manage documents  
- Generate AI responses based on your documents  
- Multi-tenant employee dashboard with authentication  
- Embeddable public chatbot UI  
- Model-agnostic LLM integration with guardrails
